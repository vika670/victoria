 📝  Описание программы
Book Tracker — это графическое приложение (GUI), предназначенное для ведения личного списка прочитанных книг. Программа позволяет систематизировать библиотеку, отслеживать объем прочитанного и быстро находить нужные произведения по критериям.

 Основные возможности:
•   Учет книг: Добавление названия, автора, жанра и количества страниц.
•   Хранение данных: Все записи автоматически сохраняются в файл books_data.json и загружаются при старте приложения.
•   Умная фильтрация: Поиск по жанру (вхождение строки) и быстрый отбор «больших» книг (более 200 страниц).
•   Валидация: Система контроля ввода предотвращает сохранение пустых записей или некорректных данных (например, текста вместо количества страниц).

 🚀  Инструкция по запуску

 Предварительные требования:
1.  Установленный интерпретатор Python версии 3.7 или выше.
2.  Библиотека Tkinter (обычно поставляется в комплекте с Python).

 Запуск:
1.  Скачайте файл с исходным кодом main.py.
2.  Откройте терминал (командную строку) в папке с файлом.
3.  Запустите приложение командой:
    
    python main.py

 🛠 Примеры использования и тестирование

 1. Добавление новой книги (Позитивный тест)
•   Действие: Введите в поля следующие данные:
    •   Название: «Преступление и наказание»
    •   Автор: «Ф. Достоевский»
    •   Жанр: «Классика»
    •   Страниц: «672»
•   Нажмите: «Добавить книгу».
•   Результат: В таблице появится новая строка, появится сообщение об успехе, данные сохранятся в JSON-файл.

 2. Валидация пустых полей (Негативный тест)
•   Действие: Оставьте поле «Автор» пустым, остальные заполните.
•   Нажмите: «Добавить книгу».
•   Результат: Программа выведет предупреждение «Все поля должны быть заполнены!», запись не будет добавлена.

 3. Валидация числового ввода (Граничный случай)
•   Действие: В поле «Страниц» введите текст «много» или отрицательное число.
•   Нажмите: «Добавить книгу».
•   Результат: Программа выдаст ошибку «Количество страниц должно быть целым числом!».

 4. Работа фильтров
•   Действие: Добавьте несколько книг с разным количеством страниц (например, 150 и 300). Активируйте галочку «Более 200 страниц» и нажмите «Применить фильтр».
•   Результат: В таблице останутся только те книги, где страниц > 200.
